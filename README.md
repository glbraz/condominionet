# condominionet
O CondomínioNet é um sistema de gerenciamento de condomínios desenvolvido em C# como projeto acadêmico. A aplicação permite cadastrar, consultar, alterar e excluir unidades, moradores e ocorrências, além de gerar relatórios e armazenar os dados em arquivos de texto, aplicando conceitos de Programação Orientada a Objetos e arquitetura MVC.
